33  // NOTE: All GPUd*() functions are also compiled on the host during GCC compilation.
34  // The GPUh*() macros are for the rare cases of functions that you want to compile for the host during GPU compilation.
35  // Usually, you do not need the GPUh*() versions. If in doubt, use GPUd*()!
